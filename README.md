# 修改自动欢迎信息显示方式脚本
适合armbian欢迎脚本和debian欢迎脚本显示的环境变量修改
## 一键脚本：
```
bash <(curl -sL https://ghp.ci/https://raw.githubusercontent.com/qljsyph/bash-script/refs/heads/main/auto-update-motd.d.sh)
```
# 欢迎信息显示内容
- **当前singbox版本**
- **singbox运行状态**
- **singbox内存占用**
- **防火墙状态**
- **cpu使用率**
- **内存使用率**
- **存储状况**
- **网络设置信息**
- **网络接口信息**
